# AcademiQ

A full-stack academic platform featuring a 3-tier role-based access system and an AI-powered question prediction feature.

## 🚀 Live Demo
🔗 [Launch AcademiQ](https://academiq-sigma.vercel.app)

## ✨ Features
* **Role-Based Access:** Secure 3-tier system for Students, Admins, and Super Admins with email verification.
* **AI Question Prediction:** Analyzes previous year question papers and syllabus to suggest high-probability exam questions.
* **Student Dashboard:** Easy browsing of uploaded question papers and exam pattern tracking by subject.

## 🛠️ Tech Stack
* **Frontend:** HTML, CSS, JavaScript
* **Backend & Logic:** Python
